# companion-module-kiloview-decoder

Companion module for **Kiloview Dxx decoders** running the [decoder-hi3536](https://github.com/kiloview/decoder-hi3536) firmware.

API routes and HTTP methods follow:

- `decoder-hi3536/multiview/src/interface/RegistHttpsRoute.cpp`
- `decoder-hi3536/multiview/res/static/js/app.js` (frontend API calls)

Default API endpoint: `http://<device-ip>:80/api/`.
